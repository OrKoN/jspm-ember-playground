Basic Ember App example using JSPM
==========

Includes plugins to compile and bundle Handlebars templates. For example:

```
  import tmpl from './templates/template.hbs!';
  var Component = Ember.Component.extend({
    layout: tmpl
  });
```

Usage
=====

1. Clone
2. `npm install`
3. `jspm init`
4. `jspm install`
6. `gulp watch`
